# 需求：主界面自动加载音乐列表

## 背景

当前 MainActivity 的交互流程要求用户每次进入界面后，必须手动点击「本地音乐」按钮才能加载音乐列表，之后才能通过底部播放栏播放音乐。这一额外步骤影响了用户体验的流畅度。

## 需求描述

### 核心逻辑

1. **已授权存储权限时**：MainActivity 创建时自动加载本地音乐列表，无需用户手动触发。
2. **未授权存储权限时**：显示「本地音乐」按钮，用户点击后触发权限申请流程；权限授予后自动加载音乐列表。

### 按钮显示/隐藏规则

| 状态 | 按钮可见性 |
|------|-----------|
| 已有存储权限，音乐列表数据加载中或已加载 | 隐藏 |
| 无存储权限，等待用户操作 | 显示 |
| 权限申请被拒绝 | 显示（用户可再次点击申请） |

### 交互流程

```
onCreate
  ├─ 已有存储权限 → 隐藏按钮 → 自动调用 getLocalSongs()
  └─ 无存储权限 → 显示按钮
                      └─ 用户点击按钮 → 申请权限
                            ├─ 授权 → 隐藏按钮 → 调用 getLocalSongs()
                            └─ 拒绝 → 按钮保持显示
```
